# AI-RESUME-SCREENING-SYSTEM
This project automates resume screening using NLP and Machine Learning.
It extracts key information from resumes, matches them with job descriptions,
and ranks candidates based on relevance.

## Technologies Used
- Python
- NLP (NLTK)
- TF-IDF
- Cosine Similarity
- Scikit-learn

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. View ranked candidates in:
   results/ranked_candidates.csv
